# 🌸 She Can Foundation Website

A modern and responsive NGO website created using **HTML** and **CSS**.  
This project is designed for the **She Can Foundation**, focusing on women empowerment, education, leadership, and social impact.

---

# 📌 Features

- Responsive Navigation Bar
- Attractive Hero/Home Section
- About Us Section
- Gallery / Impact Section
- Professional Footer
- Smooth Hover Effects
- Mobile Responsive Design
- Clean and Modern UI

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
